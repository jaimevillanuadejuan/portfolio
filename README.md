# Jaime Villanua De Juan — Portfolio

https://jaimevillanuadejuan.netlify.app/

## Overview

Personal portfolio built with React and TypeScript, featuring a modern minimalist design with smooth animations and interactive micro-interactions designed to grab recruiter attention instantly.

## Tech Stack

- React 18 + TypeScript
- SCSS with a centralized design system (variables, mixins, tokens)
- Framer Motion — scroll-triggered animations, staggered fade-ins, hover micro-interactions
- EmailJS — contact form with validation and no page reload

## Features

- Animated hero section with typewriter effect
- Fixed navbar with active section tracking via Intersection Observer
- Dynamic projects grid (responsive 1 → 2 → 3 columns) rendered from a centralized data file
- Project cards with grayscale-to-color hover, tech tags, GitHub and live site links
- Modern contact form with inline validation and smooth status messages
- Reduced motion support throughout

## Projects Showcased

- BucketTrips — AI-powered travel planning app (Next.js, NestJS, PostgreSQL, Groq LLM)
- VocalResony — Music e-commerce platform (React, Next.js, Firebase, PayPal API)
- JAMES V — Artist website (React, Firebase, Cloud Functions)
